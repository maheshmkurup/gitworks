>git init [for initializing git]

>git status [list all untracked files]

>git add -A [adding all untracked files to staging area]

>git commit -m "commit message" [add all fies from staging area to local repository]

>git clone [to get exact copy from local repository to local workspace]

>pull [for getting changes from local repository to local workspace]

>push [for getting changes from local repository to remote repository]

>git config --list [for list the configuration]

>git config global --user.name "username" [for configuring username]

>git config global --user.email "useremail" [for configuring user email]

git remote add remote_name path [to add remote]

git push remote-name branch_name [to push]
